Generación de imágenes de tomografías de pulmones sanos y con enfermedades. Proyecto

Este proyecto tiene como objetivo desarrollar un modelo GAN para generar imágenes sintéticas de tomografías pulmonares, tanto de pulmones sanos como con cáncer, con el fin de mejorar los algoritmos de detección de cáncer mediante inteligencia artificial.

Autores

  • Cotutor: Raúl del Águila Escobar
  • Cotutor: Raúl García García

Este proyecto ha sido desarrollado como Trabajo de Fin de Grado en colaboración con la Universidad CEU San Pablo, dentro del grado en Ingeniería de Sistemas de Información.

Componentes Clave

Este proyecto combina tres componentes clave: un modelo GAN para generar imágenes médicas sintéticas, un pipeline de MLOps para entrenar y mejorar continuamente el modelo, y una interfaz gráfica que permite a médicos y expertos evaluar la calidad de las imágenes generadas y proporcionar retroalimentación para su mejora.
